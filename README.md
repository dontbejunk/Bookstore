# PHP 簡易購物網站
![網站畫面](./assets/images/cover.png)

## 專案簡介 Project Overview

這是一個以 PHP 與 MySQL/MariaDB 為基礎的簡易購物網站。  
目標是練習會員登入、購物車、書籍瀏覽與訂單送出等功能的實作。

## 主要功能 Core Features

- 會員註冊與登入登出  
- 書籍瀏覽與搜尋  
- 購物車管理  
- 訂單送出
- 歷史訂單查詢
